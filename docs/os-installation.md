# Raspberry Pi OS Installation (Headless Setup)

This guide describes how to install and configure **Raspberry Pi OS Lite (64-bit)**
on a Raspberry Pi for use with the RNA-seq analysis system.

This setup uses a **headless configuration** (no desktop environment),
which reduces system overhead and improves stability.

------------------------------------------------------------------------

# Hardware Requirements

| Component | Specification |
|---|---|
| Raspberry Pi | Raspberry Pi 4 or Raspberry Pi 5 |
| microSD card | ≥ 32 GB |
| Computer | For flashing OS |
| Network | WiFi or Ethernet |

------------------------------------------------------------------------

# Install Raspberry Pi OS

Download **Raspberry Pi Imager**:

https://www.raspberrypi.com/software/

Insert your microSD card and select:

Raspberry Pi OS (Other) → Raspberry Pi OS Lite (64-bit)
